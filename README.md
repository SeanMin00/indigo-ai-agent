# Indigo AI Agent

Accessibility-focused AI agent for deaf and hard-of-hearing users.

## Overview

This project is a hackathon prototype for building a live AI agent experience with Gemini.

## Stack

- Next.js
- React
- TypeScript
- Prisma
- Supabase
- Gemini API

## Status

- Base scaffold complete
- Gemini API test route added
- Collaboration Git workflow ready
