# Decision Log

## 2026-03-25

### Stack

- `Next.js` for the web app and API routes
- `Supabase` for hosted PostgreSQL, Auth, and Storage
- `Prisma` as the application ORM
- `GitHub` for collaboration, code review, and CI

### Why

- Fast onboarding for a four-person hackathon team
- Simple deployment path with room to grow after the event
- Shared standards for schema changes, environment variables, and reviews
